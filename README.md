# GitHub Pages用アイコンファイル

このフォルダの中身をGitHub Pages用リポジトリへアップロードしてください。

アップロードするもの:
- assets フォルダ
- index.html
- config.js
- manifest.webmanifest

config.jsの中にApps ScriptのWebアプリURLを設定してください。

公開後のURL例:
https://ユーザー名.github.io/リポジトリ名/assets/apple-touch-icon.png
https://ユーザー名.github.io/リポジトリ名/assets/app-icon-192.png
https://ユーザー名.github.io/リポジトリ名/assets/app-icon-512.png
https://ユーザー名.github.io/リポジトリ名/assets/favicon-32.png
https://ユーザー名.github.io/リポジトリ名/assets/favicon-16.png
https://ユーザー名.github.io/リポジトリ名/manifest.webmanifest
